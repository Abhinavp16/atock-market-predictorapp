# Stock Price Predictor

Flutter mobile app and Node.js backend for an India-focused stock price prediction demo.

## Projects

- `flutter_app` contains the Flutter client.
- `backend` contains the Express API and mock market data.

## Notes

- The reference design folder `stitch_neural_market_predictor` is intentionally excluded from version control.
- Use `backend` on port `3000` and run the Flutter app with `API_BASE_URL` pointing to that backend.
